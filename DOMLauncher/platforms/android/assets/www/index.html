Welcome!

Thank you for using DOMLauncher, the first Cordova Powered & Crosswalk Foundational Android Launcher!

DOMLauncher provides a stock DOMod Interface (DMD/DMDi). Install custom DOMods for whole new and unique interface for your device. Create your own utilizing nothing more than HTML5/CSS3/Js!

The stock DOMod generally only appears if there is currently not a selected DOMod or there was an error loading a selection.

To install a new DOMod interface, place the unzipped folder into the DOMLauncher folder located on your device. To select a DOMod interface, choose the DOMod selection from the menu.

DOMLauncher: An Android home replacement application that loads in custom user interfaces that are written with web technologies. Included plugins fall under the DOMLauncher umbrella and are not separate from the application.
